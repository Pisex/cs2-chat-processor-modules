# [VIP] Chat
My Discord server - https://discord.com/invite/g798xERK5Y

Allows you to change the chat for a VIP player

In **groups.ini** add:
```
"Chat"		"1/0"

"Chat_Prefix"		    "value/list/custom"
"Chat_NameColor"		"value/list"
"Chat_TextColor"		"value/list"
"Chat_PrefixColor"		"value/list"
```
